Rust中的命令行应用程序

Rust是一种静态编译、快速的语言，具有强大的工具和快速增长的生态系统。
这使得它非常适合编写命令行应用程序:它们应该是小型的、可移植的和快速运行的。
命令行应用程序也是开始学习Rust或者把Rust介绍给你的团队的好方法!

使用简单的命令行界面(CLI)编写程序，对于刚接触该语言并希望对其有所感受的初学者来说是一个很好的练习。
这个话题有很多方面，但往往只有在以后才会显露出来。

本书的结构是这样的:我们从一个快速教程开始，然后你将完成一个可以使用的CLI工具。
你将接触到Rust的一些核心概念以及CLI应用程序的主要方面。接下来的几章将更详细地讨论这些方面。

在深入CLI应用程序之前，还有最后一件事:如果你在本书中发现了错误，或者想帮助我们编写更多内容，可以在CLI book repository中找到源代码。
